# チートシート

開発環境で使うコマンド、キーバインド、操作方法をまとめたもの。<br>
変更があるたびにここを更新する。


## 構成概要

| レイヤー | ツール |
|----------|--------|
| ターミナル | cmux |
| エディタ(CLI) | Neovim |
| エディタ(GUI) | VSCode |
| Git UI | codediff + gitsigns |
| シェル | zsh + starship |
| 検索 | fzf, ripgrep, fd |
| バージョン管理 | mise |
| キーリマッパー | karabiner |


## cmux (ターミナル)

ワークスペース > ペイン > サーフェス(タブ) > の3階層構造。

### ワークスペース

| キー | 操作 |
|------|------|
| `Cmd+N` | 新規ワークスペース |
| `Cmd+1-9` | ワークスペース切り替え |
| `Cmd+Ctrl+[` | 前のワークスペース |
| `Cmd+Ctrl+]` | 次のワークスペース |
| `Cmd+Shift+W` | 閉じる |
| `Cmd+Shift+R` | 名前変更 |

### ペイン

| キー | 操作 | 設定元 |
|------|------|--------|
| `Cmd+D` | 右に分割 | cmux標準 |
| `Cmd+Shift+D` | 下に分割 | cmux標準 |
| `Alt+h/j/k/l` | ペイン移動 | config |

### サーフェス (タブ)

| キー | 操作 |
|------|------|
| `Cmd+T` | 新規タブ |
| `Cmd+Shift+[` | 前のタブ |
| `Cmd+Shift+]` | 次のタブ |
| `Cmd+W` | 閉じる |


### 通知

| キー | 操作 |
|------|------|
| `Cmd+Shift+I` | 通知パネル表示 |
| `Cmd+Shift+U` | 未読通知にジャンプ |

## mise (ランタイムバージョン管理)

| コマンド | 説明 |
|----------|------|
| `mise install` | .tool-versionsに従ってランタイムインストール |
| `mise use node@20` | Node.js 20を使用 |
| `mise ls` | インストール済みランタイム一覧 |
| `mise run <task>` | タスク実行 |


## シェル (zsh)

| キー/コマンド | 説明 |
|---------------|------|
| `Ctrl+r` | 履歴検索 (fzf) |
| `z <dir>` | ディレクトリ移動 (zoxide。過去の移動先を学習) |

## Neovim 基本操作

leader = `,`

### 基本キーマップ (プラグイン非依存)

| キー | 説明 |
|------|------|
| `Esc Esc` | 検索ハイライト解除 |
| `C-^` | 前のバッファに戻る |
| `q` | 無効化 (誤操作防止) |
| `C-h/j/k/l` | ウィンドウ間移動 |
| `,x` | バッファを閉じる |
| `,c` | 現在のファイルパスをコピー |
| `,m` | Markdownプレビュー (mo) |
| `,t` | ターミナルを下に分割して開く |
| `jj` (ターミナルモード) | ノーマルモードに戻る |

### ファイルツリー (neo-tree)

| キー | 説明 |
|------|------|
| `,e` | ファイルツリーのトグル |
| `,E` | 現在のファイル位置にreveal |

neo-tree内での操作:

| キー | 説明 |
|------|------|
| `Enter` / `o` | ファイルを開く |
| `S` | 水平分割で開く |
| `s` | 垂直分割で開く |
| `t` | 新しいタブで開く |
| `a` | 新規ファイル/ディレクトリ作成 |
| `d` | 削除 |
| `r` | リネーム |
| `c` | コピー |
| `m` | 移動 |
| `q` | 閉じる |

### バッファタブ (bufferline)

| キー | 説明 |
|------|------|
| `]b` | 次のタブ |
| `[b` | 前のタブ |

### quickfix

| キー | 説明 |
|------|------|
| `]q` | 次のquickfix項目 |
| `[q` | 前のquickfix項目 |
| `,q` | quickfixリストのトグル |

### ウィンドウ・バッファ操作

| キー | 説明 |
|------|------|
| `C-w x` | 隣のウィンドウと入れ替え |
| `C-w H/J/K/L` | ウィンドウを指定方向の端に移動 |
| `C-w s` | 水平分割 (下に開く: splitbelow) |
| `C-w v` | 垂直分割 (右に開く: splitright) |
| `:b {名前}` | 任意のウィンドウで特定バッファを開く |

### ファジーファインダー (telescope)

| キー | 説明 |
|------|------|
| `,,` | ファイル検索 |
| `,r` | grep検索 (リアルタイム) |
| `,b` | バッファ一覧 |
| `,o` | 最近使ったファイル |
| `,gs` | git変更ファイル一覧 |

telescope内の操作:

| キー | 説明 |
|------|------|
| `Esc` | 閉じる (insertモードから即閉じ) |
| `Enter` | 選択して開く |
| `Tab` | 選択のトグル (複数選択) |
| `Shift+Tab` | 選択トグル (上方向) |
| `Ctrl+q` | 選択をQuickfixリストに送る |
| `Ctrl+t` | 新しいタブで開く |

### LSP

| キー | 説明 |
|------|------|
| `gd` | 定義ジャンプ |
| `gy` | 型定義ジャンプ |
| `gr` | 参照一覧 |
| `K` | ホバー (型情報表示、2回押しでウィンドウに入れる) |
| `gl` | diagnosticをフロート表示 |
| `,rn` | リネーム |
| `,ca` | コードアクション |

### 補完 (nvim-cmp)

| キー | 説明 |
|------|------|
| `Tab` | 次の候補 |
| `Shift+Tab` | 前の候補 |
| `Enter` | 確定 |
| `Ctrl+Space` | 手動で補完メニュー表示 |

### GitHub Copilot

インライン補完:

| キー | 説明 |
|------|------|
| `Tab` | サジェストを受け入れる |
| `C-]` | サジェストを閉じる |
| `M-]` | 次のサジェスト |
| `M-[` | 前のサジェスト |
| `M-\` | 手動でサジェスト要求 |
| `M-Right` | 単語単位で受け入れる |

コマンド:

| コマンド | 説明 |
|----------|------|
| `:Copilot panel` | 補完候補を最大10個一覧表示 (`Enter`で採用、`[[`/`]]`で移動) |
| `:Copilot status` | Copilotの状態確認 |
| `:Copilot enable/disable` | 有効化/無効化 |
| `:Copilot model` | モデル切り替え |

### Git (gitsigns)

| キー | 説明 |
|------|------|
| `gn` | 次の変更箇所へ |
| `gp` | 前の変更箇所へ |
| `gha` | 変更をステージング |
| `ghu` | 変更を取り消し |
| `ghp` | 変更をプレビュー |

### codediff (Git diff)

| キー | 説明 |
|------|------|
| `,gg` | diffビューを開く |
| `q` | diffビューを閉じる |
| `]c` / `[c` | 次/前のハンクへ移動 |
| `]f` / `[f` | 次/前のファイルへ移動 |
| `,b` | エクスプローラの表示/非表示 |
| `,e` | エクスプローラにフォーカス |
| `-` | ファイルのステージ/アンステージ |
| `,hs` | カーソル位置のハンクをステージ |
| `,hu` | カーソル位置のハンクをアンステージ |
| `,hr` | カーソル位置のハンクを破棄 |

コミット履歴:

| コマンド/キー | 説明 |
|---------------|------|
| `:CodeDiff history` | コミット履歴パネルを開く |
| `Enter` | 選択したコミットのdiffを表示 |
| `]f` / `[f` | 次/前のファイルへ移動 |
| `t` | サイドバイサイド/インラインの切り替え |

エクスプローラ内:

| キー | 説明 |
|------|------|
| `i` | リスト/ツリー表示の切り替え |
| `t` | サイドバイサイド/インラインの切り替え |
| `g?` | キーマップヘルプ表示 |

### 折りたたみ (foldmethod=indent)

| キー | 説明 |
|------|------|
| `zM` | 全て折りたたむ |
| `zR` | 全て展開 |
| `zc` | カーソル位置を折りたたむ (インデントされた行で実行) |
| `zo` | カーソル位置を展開 |

### コメント (Comment.nvim)

| キー | 説明 |
|------|------|
| `gcc` | 行コメントトグル |
| `gc` + モーション | 範囲コメント (例: `gcap` でパラグラフ) |

### Markdownプレビュー

| キー | 説明 |
|------|------|
| `,m` | ブラウザでプレビュー (トグル) |


## Git (コマンドライン)

| コマンド | 説明 |
|----------|------|
| `g s` | git status |
| `g l` | ログ (50件、カラー付き) |
| `g bc` | fzfでブランチを選んでcheckout |
| `g ls` | fzfでコミットを選んでshow (プレビュー付き) |
| `g fixup` | fzfでコミットを選んでfixupコミット作成 |
| `g aicommit` | AIでコミットメッセージ自動生成 (Claude Code CLI) |


## karabiner キーマップ

| キー | 動作 |
|------|------|
| `ESC` | 英数キー + ESC (IME切替) |
| `英数+h/j/k/l` | カーソルキー (上下左右) |
| `英数+a/e` | 行頭/行末 |
| `英数+u/i` | 前の単語/次の単語 |
| `英数+n/m` | BackSpace/Delete |
| `CapsLock` | Command (Mac内蔵) / Control (外部KB) |
