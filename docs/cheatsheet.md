# チートシート

開発環境で使うコマンド、キーバインド、操作方法をまとめたもの。<br>
変更があるたびにここを更新する。


## 構成概要

| レイヤー | ツール |
|----------|--------|
| ターミナル | cmux |
| エディタ(CLI) | Neovim |
| エディタ(GUI) | VSCode |
| Git UI | codediff + gitsigns |
| シェル | zsh + starship |
| 検索 | fzf, ripgrep, fd |
| バージョン管理 | mise |
| キーリマッパー | karabiner |


## cmux (ターミナル)

ワークスペース > ペイン > サーフェス(タブ) > の3階層構造。

### ワークスペース

| キー | 操作 |
|------|------|
| `Cmd+N` | 新規ワークスペース |
| `Cmd+1-9` | ワークスペース切り替え |
| `Cmd+Ctrl+[` | 前のワークスペース |
| `Cmd+Ctrl+]` | 次のワークスペース |
| `Cmd+Shift+W` | 閉じる |
| `Cmd+Shift+R` | 名前変更 |

### ペイン

| キー | 操作 | 設定元 |
|------|------|--------|
| `Cmd+D` | 右に分割 | cmux標準 |
| `Cmd+Shift+D` | 下に分割 | cmux標準 |
| `Alt+h/j/k/l` | ペイン移動 | config |

### サーフェス (タブ)

| キー | 操作 |
|------|------|
| `Cmd+T` | 新規タブ |
| `Cmd+Shift+[` | 前のタブ |
| `Cmd+Shift+]` | 次のタブ |
| `Cmd+W` | 閉じる |


### 通知

| キー | 操作 |
|------|------|
| `Cmd+Shift+I` | 通知パネル表示 |
| `Cmd+Shift+U` | 未読通知にジャンプ |

| `,h` | harpoonメニュー表示 (順序変更・削除可) |
| `,1`〜`,4` | 登録ファイルに直接ジャンプ |

### AI アシスト (codecompanion)

| キー | 説明 |
|------|------|
| `,cc` | チャットウィンドウのトグル (ノーマル/ビジュアル) |
| `,ccp` | アクション選択 (ノーマル/ビジュアル) |
| `ga` | ビジュアル選択をチャットに追加 |
| `?` | チャット内の全キーマップ表示 |

インライン編集 (Cursor Cmd+K相当):
1. コードをビジュアル選択
2. `:CodeCompanion この関数をリファクタして` と入力
3. バッファ上で直接書き換わる → `ga` で採用、`gr` で却下

ビルトインアクション (`,ccp` から選択):

| 名前 | 説明 | モード |
|------|------|--------|
| Explain code | コードの説明 | ビジュアル選択 → チャット |
| Fix code | バグ修正 | ビジュアル選択 → チャット |
| Unit tests | テスト生成 | ビジュアル選択 → インライン |
| LSP diagnostics | LSPエラーの説明 | ノーマル → チャット |
| Commit | コミットメッセージ生成 | ノーマル → チャット |

チャット内スラッシュコマンド (`/` で補完):

| コマンド | 説明 |
|---------|------|
| `/buffer` | 開いてるバッファをコンテキストに追加 |
| `/file` | ファイルを選択して追加 |
| `/symbols` | シンボル一覧を追加 (トークン節約) |
| `/fetch` | URLの内容を追加 |
| `/compact` | 履歴を要約して圧縮 (トークン節約) |

チャット内変数 (`#` で補完):

| 変数 | 説明 |
|------|------|
| `#buffer` | 現在のバッファ内容 |
| `#buffers` | 開いてる全バッファ |
| `#diagnostics` | LSPの診断情報 |
| `#diff` | git diff |
| `#selection` | 直近のビジュアル選択 |
| `#viewport` | 画面に見えてる範囲 |
| `#terminal` | ターミナル出力 |

チャット内ツール (`@` で補完): ファイル操作等のエージェント機能

## karabiner キーマップ

| キー | 動作 |
|------|------|
| `ESC` | 英数キー + ESC (IME切替) |
| `英数+h/j/k/l` | カーソルキー (上下左右) |
| `英数+a/e` | 行頭/行末 |
| `英数+u/i` | 前の単語/次の単語 |
| `英数+n/m` | BackSpace/Delete |
| `CapsLock` | Command (Mac内蔵) / Control (外部KB) |
